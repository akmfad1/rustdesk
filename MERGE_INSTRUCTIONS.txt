📍 مشکل فعلی / Current Issue
════════════════════════════════════════════════════════════════════════

دکمه "Run workflow" نمایش داده نمی‌شود چون:
- Workflow ها در branch "copilot/setup-rawdesk-client" هستند
- GitHub فقط برای branch اصلی (master) دکمه نشان می‌دهد

The "Run workflow" button doesn't show because:
- Workflows are in "copilot/setup-rawdesk-client" branch
- GitHub only shows button for main branch (master)

════════════════════════════════════════════════════════════════════════
✅ راه حل / Solution
════════════════════════════════════════════════════════════════════════

STEP 1: باز کردن Pull Request
────────────────────────────────────────────────────────────────────────

1. به این آدرس بروید:
   👉 https://github.com/akmfad1/rustdesk/pull/2

2. صفحه PR را مشاهده خواهید کرد


STEP 2: Merge کردن
────────────────────────────────────────────────────────────────────────

در صفحه PR:

1. پایین صفحه را نگاه کنید
2. دکمه سبز "Merge pull request" را پیدا کنید
3. روی آن کلیک کنید
4. در پنجره باز شده، "Confirm merge" را بزنید

┌──────────────────────────────────────────────────────────────┐
│                                                               │
│   [✓] All checks have passed                                │
│                                                               │
│   ┌─────────────────────────────────────┐                   │
│   │  🟢 Merge pull request              │  ⬅️ این را کلیک   │
│   └─────────────────────────────────────┘                   │
│                                                               │
│   This will merge 8 commits into master                      │
│                                                               │
└──────────────────────────────────────────────────────────────┘


STEP 3: رفتن به Actions
────────────────────────────────────────────────────────────────────────

1. بعد از merge موفق، به این آدرس بروید:
   👉 https://github.com/akmfad1/rustdesk/actions

2. صفحه GitHub Actions باز می‌شود


STEP 4: انتخاب Workflow
────────────────────────────────────────────────────────────────────────

در صفحه Actions:

1. از لیست سمت چپ، "Build RAWdesk (Windows 32-bit)" را پیدا کنید
2. روی آن کلیک کنید

┌──────────────────────────────────────────────────────────────┐
│  Workflows                                                    │
│  ├─ Build RAWdesk (Windows 32-bit)  ⬅️ این را انتخاب کنید   │
│  ├─ Build RustDesk Sciter                                    │
│  ├─ CI                                                        │
│  └─ ...                                                       │
└──────────────────────────────────────────────────────────────┘


STEP 5: اجرای Workflow
────────────────────────────────────────────────────────────────────────

در صفحه workflow:

1. سمت راست بالا، دکمه "Run workflow" را خواهید دید 🎉
2. روی آن کلیک کنید
3. یک dropdown باز می‌شود
4. دکمه سبز "Run workflow" در dropdown را بزنید

┌──────────────────────────────────────────────────────────────┐
│  Build RAWdesk (Windows 32-bit)                              │
│                                                               │
│  ┌──────────────────────┐                                    │
│  │  🟢 Run workflow ▼   │  ⬅️ این را کلیک کنید             │
│  └──────────────────────┘                                    │
│                                                               │
│  ┌─────────────────────────────────────────┐                │
│  │  Use workflow from                       │                │
│  │  Branch: master ▼                        │                │
│  │                                          │                │
│  │  ┌─────────────────────────────────┐    │                │
│  │  │  🟢 Run workflow                │    │ ⬅️ بعد این را  │
│  │  └─────────────────────────────────┘    │                │
│  └─────────────────────────────────────────┘                │
└──────────────────────────────────────────────────────────────┘


STEP 6: منتظر Build
────────────────────────────────────────────────────────────────────────

1. Workflow شروع به اجرا می‌کند
2. یک dot زرد 🟡 نشان می‌دهد در حال اجرا است
3. منتظر 15-20 دقیقه بمانید
4. اگر موفق شد، چک مارک سبز ✅ نشان می‌دهد


STEP 7: دانلود Artifact
────────────────────────────────────────────────────────────────────────

1. روی workflow run که تازه اجرا شد کلیک کنید
2. پایین صفحه را scroll کنید
3. بخش "Artifacts" را پیدا کنید
4. "RAWdesk-windows-x86-portable.zip" را دانلود کنید

┌──────────────────────────────────────────────────────────────┐
│  Artifacts                                                    │
│  ├─ 📦 RAWdesk-windows-x86-portable.zip  ⬇️ Download         │
│  │    Expires in 90 days                                     │
│  │    Size: ~50 MB                                           │
└──────────────────────────────────────────────────────────────┘
